# Frontend Take-Home Assignment

Take Home assignment for WorkOS interview loop.

## Getting Started

1. **Start the Backend API**:
   - Ensure you have the latest version of Node.js.
   - Run the following commands to install dependencies and start the API:
     ```bash
     cd server
     npm install
     npm run api
     ```
2. **Start the Frontend Client**:
   - Ensure you have the latest version of Node.js.
   - Run the following commands to install dependencies and start the client:
     ```bash
     cd ../client
     npm install
     npm run api
     ```
3. **Open Client in Browser**:
   - The Client will be available at http://localhost:3000
